4. What is a callback function and why will you use it?

A callback function is a function that pass into another function as a arguments and it is excuted inside the caller fuction.
The reasons of using callback function:
I can use a callback function to handle events such as mouse clicks, keyboard input, and other user interactions.
I can use callback function for resuable code that can be called from multiple places.
